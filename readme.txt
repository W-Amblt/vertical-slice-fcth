This is my submission for the COSC2636 Final Project. This was meant to be a fairly ambitious project for a action-adventure platformer ala something like Ninja-Gaiden, except with a more "Canadian focus". 

This vertical slice is very, very barebones, but has some things implemented. Coding has been implemented for extremely basic movement (walking and jumping), ground collision via tileset has been added, and some (extremely tacky looking) scenery is there. 

Progress on this project could have been done so much better in so many other ways, but this is what I have to show.

Hopefully I can improve this into something new :)
